# ru-cypress-course
Cypress. WEB UI/ API/ Visual Testing: base method, CSS selectors, Mocks, StubsI(Udemy, Sergei Semenov)

## Installation
1. Install [NodeJS](https://nodejs.org/en/)
2. Install [Git](https://git-scm.com/)
3. Clone this folder
4. Run `npm install` in the cloned folder
5. Input 'specPattern: "cypress/e2e/**/*.{js, jsx, ts, tsx}"' into cypress.config.js

## Running
1. Use `npx cypress open` for opening CyPress
2. Select "E2E Testing"
3. Choose a Browser
4. You should see tests from [e2e folder](cypress/e2e)
